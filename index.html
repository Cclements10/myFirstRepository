how to use Git

git init - creates a .git folder. contains all history for this folder

git status - tells you where everything is in terms of how it is being version and how the repository works

git add file - adds a file to be tracked.

git config --global user.name "Your Name" git config --global user.email "your_email@whatever.com"

git commit -m "message" - commits changes and saved it to the repo

git diff - show changes since last committed

it commit -am - commits ALL files repo

git blame file - shows who did what and when

git log - shows history of the logs on project

git checkout - look at project in the repo

git checkout master - return the newest version

git branch - shows what branch you are on

That is pretty much it!
